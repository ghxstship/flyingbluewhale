## C. Emergency Procedures

**Medical Incident** — Event Safety Director (radio SAFETY-1) declares; BLS medic responds within 60s; Producer holds show; 911 if BLS escalates. Designated FRD entry: south loading door. Spotify comms loop: notify Spotify lead within 5 min, written incident report within 24h.

**Fire / Smoke Alarm** — House alarm + strobes auto-trigger evac. Three egress paths: main lobby (Biscayne), south loading (alley), patio gate (north). House occupancy 605; we are at ~115 (80 + 35 crew/staff) — egress well within code. Producer is incident commander until MFR arrives. Crew muster: parking lot adjacent.

**Active Threat** — Run / Hide / Fight per MPD. Door scanners are first-line. Sworn security officer has direct MPD radio. Lockdown: green rooms + podcast studio + IT room have key-locked doors; main room shelters in podcast (interior, no windows, hard wall).

**Severe Weather** — Monitor NHC/NWS T-72. Postponement decision trigger: T-48 hurricane watch. Hurricane warning: postpone per Spotify cancellation policy. Lightning hold for any outdoor F&B service.

**Talent Indisposition** — Substitute MC pre-briefed; Spotify host has 5-min hold script; cancellation refund/comp framework predefined in talent advance; PR statement template on file with Spotify Comms.

**Power Loss** — UPS-backed for broadcast + emergency lighting ≥30 min. House PA cuts; producer escalates to acoustic-only contingency. Atmosphere preserved; show can continue.

**Crowd Surge / Capacity Breach** — Single-entry door with two-scanner pinch keeps crowd metered. M&G stanchioned single-file queue; wranglers enforce 30s/fan. If queue exceeds, M&G paused not collapsed.

**Allergic / Bar Incident** — F&B Lead carries epi-pen + allergen menu posted; non-alcoholic options 1:1 ratio. Liquor liability cert at door.

**Comms Failure** — Primary: Motorola CP200d radios — ch1 (production), ch2 (security), ch3 (medical). Backup: WhatsApp group `T1-{{SLUG}}-Showday`. Tertiary: physical runner protocol.

**Chain of Command** — Producer → Tigre EP on call → GHXSTSHIP Director of Production → Spotify LATAM Casa lead. Authority to halt show: Event Safety Director (safety), Producer (operational), Spotify lead (editorial).
