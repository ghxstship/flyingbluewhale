C. Emergency Procedures (tentpole / 2-stage / patio overlays)

Medical — 5 BLS medics + dedicated medical room + recovery zone; roving teams across 2 stages + patio + market. 400 cap + alcohol + dense floors → proactive hydration, free water, harm-reduction posture. Safety Director is medical IC; south-door FRD access; pre-event coordination with Miami FD given scale.

Fire / FX — Lasers (variance + operator cert), CO2, confetti, low-fog — all coordinated with Fire Marshal at walkthrough; laser safety officer on FX. Two-stage = two FX zones; cue-locked. Patio cooking/vendor open-flame permits handled in Executive. Egress: lobby + south + patio gate; 400 under 605 combined CO with patio in use.

Active Threat — Run/Hide/Fight; 22 security incl. multiple sworn officers w/ MPD radio; unified command post; show-stop "house up / music cut" cue carded at both FOH positions + all dept heads; lockdown via key-locked rooms; market vendors briefed.

Crowd / 2-stage flow — Highest risk is stage-swap migration (crowd moving main↔patio). Staggered set times + DJ bridges keep flow, not crush; one-way-preferred circulation; security manages the pinch between rooms; hard 400 cap via scan reconciliation, one-in-one-out at cap; patio sub-count monitored separately.

Severe Weather — Patio stage + market are weather-exposed: wind-rated structures, lightning protocol (move patio program inside / hold), rain plan compresses to indoor stage. NHC/NWS T-72; postpone trigger T-48 hurricane watch; multi-truck storm load-out contingency.

Power Loss — Generators back the festival rig (Production line); UPS for emergency lighting/comms; per-stage distro isolation so one stage fault doesn't dark the other; show-stop protocol on primary fault.

Noise (MiMo residential) — Two SPL monitoring points (indoor + patio property line) per sound permit; cardioid sub arrays aimed off-neighborhood; patio program curfew earlier than indoor; dedicated community-relations contact on call all night (ties to the Discovery-phase neighborhood MOU).

Vendor / Market — Each vendor carries own permits + insurance (verified in Advance); fire extinguishers per stall; cash/POS security; allergen labeling enforced.

Comms — Motorola CP200d: ch1 production, ch2 security, ch3 medical, ch4 main stage, ch5 patio stage, ch6 bar/market/ops. Backup WhatsApp T4-{{SLUG}}-Showday. Show-stop is an all-channel call from the unified command post.
