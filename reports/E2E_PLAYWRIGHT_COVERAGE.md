### Environment note тАФ local dev-server load ceiling

The local Turbopack dev server wedges under sustained cold-compile load (observed at ~500+ routes in the HTTP sweep, and when chaining ~20+ creates in one run). Consequently each spec **file** is green when run on its own, but a single `npm run e2e` of _all_ batches can wedge the server partway. In CI (per-shard fresh servers + `retries: 2`) this is absorbed. For local verification, run batches individually: `npx playwright test console-modules-b5`. The uncovered remainder are routes that hard-require a pre-existing related record chosen from a dropdown (agency toursтЖТtalent, rentalsтЖТequipment, pay-apps/po-change-ordersтЖТPO, prequalificationтЖТvendor+questionnaire, submittalsтЖТvendor, MSAsтЖТcrew, subscriptionsтЖТparty, maintenanceтЖТtarget, takeoffsтЖТsite-plan, site-plansтЖТcomposite code) тАФ best driven by fixture-seeded specs тАФ plus a couple of map/composite-input forms.
